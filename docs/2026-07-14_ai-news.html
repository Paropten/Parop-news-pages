1. Excel：フィルター後の表示セルだけをコピーする

何が便利か：非表示行を含めず、画面に見えている結果だけを別表へ貼り付けられます。

  • 手順：範囲を選択 → Alt + ; → Ctrl + C → 貼り付け。
  • 注意点：結合セルがある表では選択が崩れる場合があります。
  • 活用メモ：抽出後の対象一覧や確認表を作る際に便利です。

2. Windows 11：ファイルパスをすぐコピーする

何が便利か：共有フォルダや資料の場所を正確にメールやチャットへ貼れます。

  • 手順：エクスプローラーでファイルを選択 → Shiftを押しながら右クリック → 「パスのコピー」。
  • 注意点：相手が同じ共有場所へアクセスできるか確認します。
