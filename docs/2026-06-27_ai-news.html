今日の便利Tips

Excel

Tips 1. テーブル化でフィルターと集計を安定させる

何が便利か

範囲をテーブル化すると、行を追加しても数式やフィルターの対象が自動で広がり、月次集計や一覧管理が崩れにくくなります。

手順

  1. 一覧表の中をクリックする。
  2. Ctrl + T を押す。
  3. 先頭行を見出しとして使用する場合はチェックを入れて OK を押す。
  4. テーブルデザインから名前を付ける。

注意点

見出しが空欄だと自動で列名が付くため、後から数式で参照しにくくなります。先に列名を整えてから実行すると安全です。

活用メモ

問い合わせ一覧、作業予定表、利用料管理表など、行が増える表は最初にテーブル化しておくと後の修正が減ります。

Windows 11

Tips 2. クリップボード履歴で直前にコピーした文字を再利用する

何が便利か

複数のURL、メール文、コマンド、エラー文を一時的に保持できるため、コピーし直しの手間を減らせます。

手順

  1. Windowsキー + V を押す。
  2. 初回は「有効にする」を選ぶ。
  3. 履歴から貼り付けたい内容をクリックする。
  4. よく使うものはピン留めする。

注意点

パスワードや個人情報をコピーした場合は履歴に残る可能性があります。共有PCやリモート作業では、履歴の削除を意識してください。

活用メモ

障害対応時のログ断片、メール返信の定型文、確認用URLを一時的に扱うときに便利です。
