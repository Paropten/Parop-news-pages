今日の便利Tips

Tips 1：Windows 11で画面の一部だけをすぐ文字認識する

Snipping Toolで範囲を切り取り、「テキスト アクション」を選ぶと画像内の文字をコピーできます。

  • 手順：Windows + Shift + S → 範囲選択 → 通知を開く → テキスト アクション。
  • 注意点：表や小さい文字は誤認識するため、貼り付け後に確認する。
  • 活用メモ：エラー画面やPDF画像から短い文字列を抜き出すときに便利です。

Tips 2：Excelで表示中のセルだけをコピーする

フィルター後の表をコピーするときは、Alt + ; で可視セルだけを選択できます。

  • 手順：範囲選択 → Alt + ; → Ctrl + C。
  • 注意点：非表示行を含めたい場合は通常コピーを使う。
  • 活用メモ：抽出結果だけを別シートやメールへ貼り付けるときに便利です。
